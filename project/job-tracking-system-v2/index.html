Project Overview

In today's fast-paced world, managing projects and tracking every detail can be a complex and overwhelming task. Imagine a solution that simplifies these processes, makes them more organized, and effortlessly boosts your team's productivity.

Welcome to JOB TRACKING SYSTEM V2, a comprehensive, next-generation platform designed to revolutionize how you manage your work. This system promises to move you beyond traditional methods and help you reach new levels of efficiency.

With its comprehensive feature set and modern technology stack, JOB TRACKING SYSTEM V2 is not just a tool; it's a strategic partner for your team's success. It's designed to make project management more intuitive, collaborative, and efficient, allowing your team to focus on what matters most: building and delivering great work.

Core Modules and Features

User Management & Authentication

Complete user management suite with registration, login, secure profile updates, and invitation system.

Users can easily create accounts with email verification and password reset functionality.

Profile updates require email verification through one-time codes for enhanced security.

Administrators can invite new users via email with automatic project assignment.

Role-based access control ensures users only see and interact with appropriate data.

Project Module

Create, list, update, and delete projects with comprehensive project management capabilities.

Add and remove individual members from projects with customizable permission levels.

Create specific teams within projects to organize members and streamline collaboration.

Define project-specific task statuses to perfectly match your unique workflow requirements.

Track project progress with visual indicators and milestone tracking features.

Task Module

Granular control over work items with full CRUD operations for comprehensive task management.

Easily assign tasks to specific users within a project with due dates and priorities.

Automated backlog management moves unassigned tasks to the project backlog automatically.

Add, update, and delete comments on tasks to facilitate communication and collaboration.

Create and manage subtasks to break down complex tasks into manageable components.

Backlog Module

Dedicated space for tasks that aren't yet scheduled, providing a repository for future work.

Automatic task handling moves tasks created without a sprint assignment directly to the backlog.

Tasks removed from sprints are automatically returned to the project backlog for reassignment.

Easily add, remove, and delete tasks from the backlog with intuitive drag-and-drop functionality.

Prioritize backlog items to ensure the most important tasks are scheduled for upcoming sprints.

Sprint Module

Ideal for teams using Agile methodologies with comprehensive sprint management capabilities.

Create, list, and delete sprints on a per-project basis with customizable durations.

Easily add and remove tasks from sprints with visual feedback and progress tracking.

Assign team members to specific sprints to clarify responsibilities and resource allocation.

Track sprint progress with burndown charts and completion metrics for data-driven insights.

Kanban Module

Visual and interactive way to manage your workflow with dynamic, customizable Kanban boards.

View all tasks across all sprints or focus on a single sprint with flexible filtering options.

Change a task's status by dragging and dropping it between columns for intuitive workflow management.

Customize board columns to match your team's specific workflow stages and processes.

Set work-in-progress limits to prevent bottlenecks and maintain a steady flow of tasks.

Weekly Module

Specialized module to track time and progress on a weekly basis with comprehensive reporting.

Users can enter their hourly work details for each task and project on a weekly basis.

Create, update, and delete weekly entries with intuitive time tracking interfaces.

Detailed weekly board screen provides a visual breakdown of hourly work by task and project.

List view shows a summary of all weekly entries with filtering and sorting capabilities.

File Management

Handle file uploads with MinIO object storage, compatible with Amazon S3 for scalability.

Attach files to tasks, projects, and comments with drag-and-drop functionality.

Preview common file types directly in the interface without downloading.

Set file permissions to control who can view, edit, and download attachments.

Track file versions and maintain a history of changes for important documents.

Security

Robust security with JWT authentication for secure, stateless user sessions.

Spring Security framework protects against common vulnerabilities like CSRF, XSS, and SQL injection.

API-based logging captures every request with request/response bodies and headers for thorough auditing.

Role-based access control ensures users can only access authorized data and functions.

Email verification for sensitive actions like password changes and account updates.

Getting Started

Frontend Setup

  1. Clone the repository:
    git clone https://github.com/celalaygar/JOB-TS-V2-FE.git
  2. Navigate to the project directory:
    cd JOB-TS-V2-FE
  3. Install dependencies:
    npm install
  4. Configure environment variables:
    cp .env.example .env.local
  5. Run the development server:
    npm run dev

Backend Setup

  1. Clone the repository:
    git clone https://github.com/celalaygar/JOB-TS-V2-BE.git
  2. Navigate to the project directory:
    cd JOB-TS-V2-BE
  3. Build the project with Gradle:
    ./gradlew build
  4. Configure environment variables:
    cp .env.example .env
  5. Run the application:
    java -jar build/libs/job-ts-v2-be-0.0.1.jar
